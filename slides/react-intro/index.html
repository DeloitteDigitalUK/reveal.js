Webapps

React in a Nutshell

  • Library for Views (Not a framework)
  • Declarative. Just describe your page
  • Breaks pages into components
  • One way data flows
  • (Built by Facebook)

Library for Views

React takes your data, and draws a webpage based on it.

Out of the box, it does not dictate how you store data, test your app, handle routing, or anything else.

Just Describe your Page

When your data is xyz, how should it look? And when it is abc, how should it look?

You don't worry about changing from xyz to abc. React handles that for you. Simple!

Break Pages into Components

Each component is responsible for one thing. Components can have child components, to move functionality out into its own.
